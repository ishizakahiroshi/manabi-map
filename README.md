# Manabi Map

住所を起点に、通える高校を地図で見て、親子で比較・記録・検討する進路管理サービスです。

- 群馬県版 MVP を 2026-08-01 リリース目標で開発中
- Web 完結、PWA 対応予定
- コードは AGPL-3.0-or-later、公開データは CC BY-SA 4.0 での提供を予定

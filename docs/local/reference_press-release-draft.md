報道関係者各位

群馬発・住所を入れると「通える高校」が地図に表示される無料サービス「Manabi Map（まなびマップ）」関東7都県1,362校対応で本格稼働のご案内

〔市区町村名〕在住のソフトウェア開発者・石坂〔氏名を確認〕は、2026年7月、中学生と保護者向けの高校選び支援サービス「Manabi Map（まなびマップ）」の関東7都県1,362校対応版を公開しました。自宅の住所を入力すると、通える範囲の高校が地図上にピンで表示され、徒歩・自転車・バスなどでの通学時間の目安を確認できます。気になる学校はお気に入りとして保存でき、文化祭や学校説明会の見学メモ、通学経路のメモを家族で共有しながら進路の家族会議を進められます。

サービスは 2026年7月上旬に群馬県版（全日制など83校）としてまず公開し、その後およそ1週間で東京・神奈川・埼玉・千葉・茨城・栃木・群馬の関東7都県・1,362校まで対応地域を拡大しました。利用は無料で、アプリのインストールは不要、スマートフォンやパソコンのブラウザからそのまま使えます。ログインは Google と LINE の2種類に対応し、ログインせずに地図検索だけを試すこともできます。

開発の背景
高校選びの情報は偏差値順の一覧サイトに偏りがちで、親子が最初に知りたい「自宅から現実的に通えるのはどの学校か」に答える地図がありませんでした。Manabi Mapは学校の序列づけやランキングではなく、家庭ごとの進路検討を整理する「学校選びの地図ノート」を目指しています。掲載する学力情報は他サイトからの転載を行わず、公的資料に基づく独自の参考値（Manabi Map 独自推計）のみを扱う方針です。また、中高生と保護者が使うサービスであることから、掲載する広告は進路・教育に関係するものに限定し、無差別配信のディスプレイ広告は載せない方針としています。

プログラムはオープンソースソフトウェア（コード AGPL-3.0 / データ CC BY-SA 4.0）として公開しており、地域の教育関係者や自治体・NPOとの連携も歓迎しています。
